EXPLICAACION DEL USO DEL ENUM:

Los enumeradores (enum) se utilizan para crear un conjunto de nombres constantes que representan valores fijos. 
En lugar de usar números (0, 1, 2) o textos libres ("Pendiente", "Procesando"), que pueden escribirse mal, usas un enum.
